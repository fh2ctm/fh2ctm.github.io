Automatic Light/Dark Switch

Vim does not support native light/dark switch based on your terminal background / system theme.

Fortunately, if you are using macOS, there is dark-notify to help! You can run dark-notify -e to get the current app theme, either light or dark.

So, you can set a variable from your shell,

export BG=$(dark-notify -e)

and read it in your vimrc,

" .vimrc
if $BG[0] == 'l'
    set bg=light
else
    set bg=dark
endif
