Tips and tricks for using vim.

Config

You can get my configuration files here. I’ll explain here what my config does.

if $BG[0] == 'l'
    set bg=light
else
    set bg=dark
endif

this is explained in the next section.

colorscheme wildcharm
filetype indent plugin on
syntax enable

select colorscheme and enable syntax highlighting and filetype plugins.

let g:context_highlight_tag = '<hide>'

hide highlight tag for context.vim.

let g:mapleader = "\ "
let g:maplocalleader = "\\"

set leader keys.

let g:fzf_colors = ...

adjust fzf window colours to match colorscheme.

let g:fzf_vim = { 'preview_window' : [] }

disable fzf preview window.

let g:snipMate = { 'snippet_version' : 1 }

snipmate options.

let no_plugin_maps = 1

don’t use filetype plugin mappings. The rest of the file are keymaps and options.

Appearance

Automatic Light/Dark Switch

Vim does not support native light/dark switch based on your terminal background / system theme.

Fortunately, if you are using macOS, there is dark-notify to help! You can run dark-notify -e to get the current app theme, either light or dark.

So, you can set a variable from your shell,

export BG=$(dark-notify -e)

and read it in your vimrc,

" .vimrc
if $BG[0] == 'l'
    set bg=light
else
    set bg=dark
endif

et voilà!

Alignment

Sometimes you’d want to align column blocks in your code. How do you do this?

I’ve found vim-easy-align to be quite useful. For that, I’ve mapped - to the alignment key:

nnoremap - <Plug>(EasyAlign)
xnoremap - <Plug>(EasyAlign)

now suppose I want to align the types in the following:

let favouriteNumbers: [Int]
let name: String

with my cursor at the beginning of the first line, I just enter Vj-2 <CR> and it will be aligned:

let favouriteNumbers: [Int]
let name:             String

very nice.
