Goal

The goal is to create a keyboard layout that is simple to learn, structural, and as close to the most common ANSI QWERTY as possible.

Differences from ANSI QWERTY

There are only 3 keys that are different with ANSI QWERTY:

  1. The W key
  2. The apostrophe ' key
  3. The grave ` key

In more detail,

Original New New under shift New under alt
double vé w e accent aigu é e accent aigu É none
apostrophe ' accent circonflexe ^ accent tréma ¨ none
grave ` accent grave ` apostrophe ' guillemet anglais double "

the three accents (`, ^, and ¨) are dead keys, meaning if you would like to type è for example, you would input `e.

In addition, the following positions have additional code points under alt,
