Spring 2024

Supervised by Ashwin Nayak.

We attempt to improve the \(f\)-query complexity deterministic classical algorithm for the general Hidden Subgroup Problem (HSP).

The setup for the HSP is:

  • Let \(G\) be a finite group.
  • Let \(H < G\) be an unknown proper subgroup.
  • Let \(f: G \to S\) be an oracle called the hiding function, where \(S\) is an finite set, with the property \[f(a) = f(b) \iff aH = bH\] that is, \(f\) is constant on the cosets of \(H\).
  • \(f\) is called injective if \(H\) is trivial, and balanced otherwise.

The goal of the HSP is:

  • Determine which case is \(f\).
  • Furthurmore, if \(f\) is balanced, find the hidden subgroup \(H\).
