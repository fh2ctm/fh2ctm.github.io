I discuss different input methods for the Chinese language.

Zhuyin (Bopomofo)

A phonetic system developed by the Republic of China (Taiwan) in the 20th century.

I got interested in it because I would like to type Traditional Chinese. Seems like Taiwan is still using Zhuyin (instead of the IMO vastly superior ulpb). so I thought I'd try it out. It was very uncomfortable at first, but I think I got used to it.

What makes it cool is that in some sense, it is the only "native" Chinese phonetic writing system. However, there are some design choices that I have issues with.

Syllables take one of the following structures.

  • vowel
  • semivowel + vowel
  • consonent + vowel
  • consonent + semivowel + vowel

When a consonant glyph appear by itself, it is considered a full syllable with vowel ɨ.

There are also tonal marks, but they are full-width and placed right after each syllable construction, and I find it too ugly to use.

Here are some examples.

  • 我觉得 wojuede ㄨㄛㄐㄩㄝㄉㄜ
  • 雙拼 shuangpin ㄕㄨㄤㄆㄧㄣ
  • 更好啊 genghaoa ㄍㄥㄏㄠㄚ

The glyphs follow (some of the) Chinese calligraphy traditions, so in some ways it is more suitable. However, it has some of the shortcomings as Pinyin, namely: it can be ambiguous for certain phrases like 西安/先, both spelled xian and ㄒㄧㄢ without using special glyphs or whitespaces.
